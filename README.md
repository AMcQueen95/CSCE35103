# CSCE35103

Software Engineering lazer tag project

## Project Setup

### Requirements:

- Java 21 (or higher)
- Maven
- Node.js

### Backend Setup (Java/Spring Boot)

1. Clone the repository.
2. Navigate to the `backend` folder.
3. Run the following commands to build and run the backend:
   ```bash
   mvn clean install
   mvn spring-boot:run
   ```
