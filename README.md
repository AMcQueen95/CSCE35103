# CSCE35103

Software Engineering lazer tag project

## Project Setup

### Requirements:

- Java 21 (or higher)
- Maven
- Node.js

## Instructions to Run Project

1. Navigate to the `backend` folder.
2. Run the following commands to build and run the backend:
   ```bash
   mvn clean install
   mvn spring-boot:run
   ```
3. Navigate to the frontend folder
4. Run the following:
   npm start
